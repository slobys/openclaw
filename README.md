# 飞牛NAS一键部署OpenClaw

## 项目主要实现功能
* 一键部署
* 一键卸载
  
### 进入home目录
```bash
cd /home/naiyou
```
### 拉取项目
```
curl -L https://raw.githubusercontent.com/slobys/openclaw/main/openclaw-fnos-menu.sh -o openclaw-fnos-menu.sh
```
### 添加权限并执行脚本
```bash
chmod +x openclaw-fnos-menu.sh
./openclaw-fnos-menu.sh
```
